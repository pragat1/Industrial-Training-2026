## Zero-Shot Prompting

Performed tasks without providing examples.

### Exercises

1. Text Summarization
2. English to Hindi Translation
3. Python Program Generation
   - Factorial using Recursion
   - Time Complexity Explanation
   - Code Comments

## One-Shot Prompting

Provided a single example before generating output.

### Exercises

1. Sentiment Analysis
   - Positive
   - Negative
   - Complaint

2. English to SQL Conversion

Example:

```sql
SELECT * FROM employees
WHERE salary > 50000;
```

3. Information Extraction

Converted normal text input into structured JSON output.

## Few-Shot Prompting

Provided multiple examples before generating output.

### Exercises

#### Animal Classification

Input Examples:

- Tiger → Animal
- Rose → ?

Classified emails into predefined categories using examples.

#### Logical Reasoning

Problem:

Task:

- Calculate profit percentage
- Generate step-by-step reasoning
